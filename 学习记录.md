## 学习内容

### 基础知识点

#### 1. 变量与打印
- **变量定义**：使用 `变量名 = 值` 的格式，如 `name = "小明"`
- **打印输出**：使用 `print()` 函数，如 `print("Hello, Python!")`
- **字符串拼接**：使用 f-字符串，如 `print(f"我的名字是{name}")`

#### 2. 用户输入与类型转换
- **获取输入**：使用 `input()` 函数，如 `name = input("请输入你的名字：")`
- **类型转换**：使用 `int()`、`float()` 等函数，如 `num = int(input("请输入一个数字："))`

#### 3. 算术运算
- **基本运算**：`+`（加）、`-`（减）、`*`（乘）、`/`（除）
- **整数除法**：`//`（取整）、`%`（取余）

#### 4. 条件判断
- **if 语句**：`if 条件:` 后跟缩进的代码块
- **多分支**：`if`、`elif`、`else` 结构
- **条件表达式**：使用 `>`、`<`、`==`、`!=` 等运算符

#### 5. 循环
- **for 循环**：用于遍历序列，如 `for item in list:`
- **while 循环**：基于条件重复执行，如 `while count <= 5:`
- **range() 函数**：生成数字序列，如 `range(1, 6)` 生成 1-5

#### 6. 函数
- **函数定义**：使用 `def 函数名(参数):` 格式
- **函数调用**：使用 `函数名(参数)` 格式
- **返回值**：使用 `return` 语句返回结果

#### 7. 字符串方法
- **upper()**：转换为大写
- **lower()**：转换为小写
- **capitalize()**：首字母大写
- **len()**：计算字符串长度

#### 8. 列表操作
- **创建列表**：使用 `[]`，如 `fruits = ["苹果", "香蕉"]`
- **添加元素**：使用 `append()` 方法，如 `fruits.append("橘子")`
- **删除元素**：使用 `del` 语句，如 `del fruits[0]`
- **检查元素**：使用 `in` 运算符，如 `"苹果" in fruits`
- **列表长度**：使用 `len()` 函数，如 `len(fruits)`

### 代码示例

#### 1. 打印语句
```python
print("Hello, Python!")
```

#### 2. 变量与打印
```python
name = "小明"
age = 18
print(f"我的名字是{name}，今年{age}岁")
```

#### 3. 获取输入
```python
name = input("请输入你的名字：")
print(f"你好，{name}！欢迎学习Python！")
```

#### 4. 计算两数之和
```python
num1 = int(input("请输入第一个数字："))
num2 = int(input("请输入第二个数字："))
sum_result = num1 + num2
print(f"两数之和是：{sum_result}")
```

#### 5. 条件判断
```python
num = int(input("请输入一个数字："))
if num > 0:
    print("这是一个正数")
elif num < 0:
    print("这是一个负数")
else:
    print("这是零")
```

#### 6. 循环
```python
# for循环遍历列表
fruits = ["苹果", "香蕉", "橘子", "葡萄"]
for fruit in fruits:
    print(fruit)

# while循环打印1-5
count = 1
while count <= 5:
    print(count)
    count += 1

# 计算1到n的和
n = int(input("请输入一个整数："))
sum_result = 0
for i in range(1, n+1):
    sum_result += i
print(f"1到{n}的和是：{sum_result}")
```

#### 7. 函数
```python
# 定义无返回值的函数
def greet(name):
    print(f"你好，{name}！")
greet("小明")

# 定义有返回值的函数
def is_even(num):
    if num % 2 == 0:
        return True
    else:
        return False
print(is_even(4))  # 输出 True
print(is_even(5))  # 输出 False
```

#### 8. 字符串方法
```python
s = "Hello World"
print("大写：", s.upper())      # HELLO WORLD
print("小写：", s.lower())      # hello world
print("首字母大写：", s.capitalize())  # Hello world
print("长度：", len(s))        # 11
```

#### 9. 列表操作
```python
# 购物清单管理
shopping_list = []
shopping_list.append("苹果")
shopping_list.append("香蕉")
shopping_list.append("牛奶")
print("当前购物清单：", shopping_list)  # ['苹果', '香蕉', '牛奶']
del shopping_list[0]
print("删除第一个物品后：", shopping_list)  # ['香蕉', '牛奶']
if "香蕉" in shopping_list:
    print("香蕉在购物清单中")
else:
    print("香蕉不在购物清单中")
```

## 学习总结

通过本次学习，掌握了Python的基础语法和核心概念，包括：

1. **变量与数据类型**：学会了如何定义变量、存储数据，并使用不同的数据类型（字符串、整数等）。

2. **输入输出**：掌握了如何获取用户输入并进行类型转换，以及如何打印输出结果。

3. **控制流**：理解了条件判断（if语句）和循环（for、while）的使用方法，能够根据不同条件执行不同代码。

4. **函数**：学会了如何定义和调用函数，以及如何返回函数结果，提高代码的复用性。

5. **数据结构**：掌握了列表的基本操作，包括添加、删除、查看元素等。

6. **字符串操作**：学会了使用字符串方法处理文本数据，如大小写转换、长度计算等。

这些基础知识是学习Python的重要基石，为后续学习更复杂的内容（如面向对象编程、模块导入等）打下了基础。通过不断练习和应用，可以逐步提高Python编程能力。
